# Smart Calculator & Utility Suite (Python)

A beginner-friendly Python program that combines multiple calculator and utility functions in one menu-driven application.

## Features
- Find greatest of 3 numbers
- Addition
- Subtraction
- Multiplication
- Division
- Even or Odd check
- Swap two numbers
- Simple Interest calculation

## Requirements
- Python 3.x

## How to Run
1. Save the code as `smart_calculator.py`
2. Open terminal or command prompt
3. Run:
   python smart_calculator.py

## Learning Concepts
This project helps practice:
- Functions
- Loops
- Conditional statements
- User input/output
- Basic arithmetic operations

## Purpose
Perfect for beginners who want to improve Python programming skills while building a practical mini-project.
